# bugset-dbupgrade

#### 项目介绍
- bugset-dbupgrade是一款数据库升级组件
- 支持MySQL、Oracle数据库
- 支持单节点单数据库，多节点单数据库的升级

#### 软件架构
```
bugset-dbupgrade
├─dbupgrade-core 数据库升级核心功能
│
├─dbupgrade-spring spring整合组件
│ 
├─dbupgrade-spring-boot-starter 配置信息

```
