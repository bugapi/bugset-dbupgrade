# bugset-dbupgrade

#### 项目介绍
- bugset-dbupgrade是一款关系型数据库升级组件
- 支持MySQL、Oracle数据库
- 支持DML和DDL的升级
- 支持单节点单数据库，多节点单数据库的升级
- 支持升级版本的记录和回溯

#### 软件架构
```
bugset-dbupgrade
|
├─dbupgrade-core 数据库升级核心功能
│
├─dbupgrade-spring spring整合组件
│ 
├─dbupgrade-spring-boot-starter springboot自动配置类

```
